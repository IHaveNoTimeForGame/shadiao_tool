## 目标

自动配图工具

### 原理


在这个中文互联网内容氛围都被带向了奇怪的方向的时代。 博客文案如果没有沙雕表情包，都没办法看下去。 每次写完博客和文案都需要配图加表情包，好麻烦。 那如果有工具可以自动添加那该多好!

于是乎，本着外事问谷歌，内事问百度，某事问天涯的原则，上网搜索了一通，没有发现有此类的工具，可以直接使用。 无奈只好自己动手丰衣足食，撸了一个。

大概思考了片刻，确定了思路

1. 无外乎就是对一篇文章分段，确定多少文字配一张表情包
2. 提取每一个分段的关键字，确定放什么表情包
3. 然后把文字和图片混合就行

关键词提取那就用Jieba。
获取表情包图片就用万能的爬虫。
图文混合，想到的办法就是造word文档
打包用pyinstaller。

### 使用
选择文本文件，用换行表示，多长一段需要配一张图。
